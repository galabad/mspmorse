mm.c is a template msp430 launchpad program, test compiled on linux debian, that flashes the launchpad's led in morse code. mm.cl is a snippet of lisp code that converts a string into a list of morse code functions for use in mm.c

You can install common lisp and an editor here:
http://common-lisp.net/project/lispbox/

You can learn about common lisp here:
http://www.gigamonkeys.com/book
